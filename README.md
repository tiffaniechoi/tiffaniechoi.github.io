# Tiffanie Choi — Personal Portfolio

Hi! This is the source code for my personal portfolio site. I built it from scratch with vanilla HTML, CSS, and JavaScript. 
**Live:** https://tiffaniechoi.github.io

---

## About the Site

I'm a Product Manager II at Microsoft with a software engineering background. This site is where I share my experience, projects, and a little bit about who I am outside of work.

It features scroll-triggered animations, an active nav highlight as you scroll through sections, and a Beyond Work section with some of my favorite photos.

---

## Project Structure

```
/
├── index.html              # Entry point (required in root for GitHub Pages)
├── src/
│   ├── style.css           # All styles and animations
│   └── main.js             # Scroll reveal, nav behavior, skill pills
├── projects/
│   └── 2018_central_park_squirrels.html  # Data analysis project (Jupyter export)
└── assets/
    ├── favicon.svg
    └── photos/
        ├── flaticon/       # Tech stack icons
        ├── jobs/           # Company logos
        ├── polaroids/      # Personal photos
        ├── profile/        # Profile photo + emoji favicon
        ├── projects/       # Project screenshots
        └── school/         # University logo
```
